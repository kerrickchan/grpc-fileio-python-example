# GRPC File Upload and Download

grpc of file upload and download in python

# Environment

* python 3.11

# Install

```bash
python -r requirements.txt
```

# Build

```bash
chmod + x ./build-protos.sh
./build-protos.sh
```

# Start Server

```bash
python ./app/server.py
```

# Test Client

```bash
python ./app/client.py
```
